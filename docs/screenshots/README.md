# Скриншоты

Положите сюда обезличенные скриншоты приложения под этими именами — они подставятся в основной README:

- `app.png` — главный экран интерфейса (кнопка запуска, статус шагов).
- `logs.png` — вкладка «Логи» с выводом в реальном времени.
- `telegram.png` — пример уведомления в Telegram (успех или ошибка со скриншотом).

## Перед публикацией обязательно скройте:

- название и реквизиты компании-клиента (ИНН, ФИО директора, наименование организации);
- суммы, налоговые базы, периоды с реальными данными;
- логин/URL баз 1С в адресной строке или настройках;
- токен и chat_id Telegram.

Замазать можно прямоугольником/блюром в любом редакторе. Лучше один аккуратный обезличенный скрин, чем «как есть» с данными клиента.
